# Project 2 – Discount vs Profit Analysis

## Objective

Analyze how discount levels impact profitability across categories, regions, and time.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Scatter Plot
* Heatmap
* Slicers

## Key Insights

* Higher discounts often reduce profit
* 30%+ discount band has many loss orders
* Some categories over-discount without returns
